LightningBot
# Requirements
  Make sure that node is at least version 6 or greater.
` $ node --version ` => v6.0.0
# Install
` $ git clone https://github.com/Youtubrer/LightningBot.git `
` $ npm run setup YOUR_TOKEN_HERE`
_I will add email in the fututure, shouldn't take long_

` $ node index.js `
Tada!
